# PopX React Assignment

This project is a React JS internship assignment built from the provided Adobe XD design.

The application is a responsive mobile-style UI with multiple screens including Landing Page, Login, Signup, and Profile page. The project focuses on clean UI implementation, reusable React components, routing, and responsive design.

## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/Gajanan06/PopX-Task
```

Move into the project folder:

```bash
cd popx-app
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
